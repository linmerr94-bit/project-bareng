  1. User fills form (topic, email, message)

  3. Fetch API POST to /api/care-hub/submit/

     {
       "topic": "checkout",
       "email": "user@example.com",
       "message": "Question/issue description"
     }

  ✓ Success: Display reference code (green alert)

     - Auto-close after 3 seconds

  ✓ Error: Display error message (red alert)
     - Message: "❌ Error: [backend error]"
     - Form remains open for retry
     
  ✓ Network Error: Display error (red alert)
     - Catch and display fetch errors
     - User can retry

  • View: submit_care_hub_inquiry() in views.py
