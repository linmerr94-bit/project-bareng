### Color Palette (100% Consistent)
```
PRIMARY BRAND:
  Deep Purple: #4c1d95
  Indigo: #6366f1
  
BACKGROUNDS:
  White: #ffffff (surfaces)
  Light Gray: #f8f9fa (page background)

  Primary Dark: #1a202c
  Secondary Gray: #64748b
  Light Border: #e2e8f0
  
SEMANTIC:
  Success: #22c55e (green)
  Warning: #fbbf24 (yellow)
  Danger: #ef4444 (red)
```

### Typography (100% Consistent)
```
Font Stack: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

Hierarchy:
  Hero/Main Title: 32-40px, Bold/700
  Page Title: 24-32px, Bold/700
  Section Title: 20-24px, Bold/700
  Heading: 16-18px, SemiBold/600
  Body: 14-16px, Regular/400
  Caption: 12-14px, Medium/500
```

### Components (100% Consistent)
```
✅ Gradient Buttons
   - Primary: Purple → Indigo gradient
   - Hover: Scale up + shadow boost
   - Transition: 0.3s smooth

✅ Input Fields
   - Border: 2px light gray
   - Focus: Indigo border + subtle box-shadow
   - Radius: 8px rounded

✅ Cards
   - Background: White
   - Border: 1px light gray or none
   - Shadow: Soft 0 2-4px rgba(0,0,0,0.08)
   - Hover: Slightly larger shadow

✅ Floating Chat Widget
   - Fixed bottom-right position
   - 60x60px circle
   - Gradient purple background
   - Scale 1.1 on hover

✅ Glassmorphism Elements
   - Semi-transparent white background
   - Subtle backdrop blur
   - Light border for definition
```

### Responsive Breakpoints (100% Consistent)
```
DESKTOP (>1024px)
  - Full 2-column layouts where applicable
  - 4-column product grids
  - All hover states active
  - Sticky elements functional

TABLET (768px-1024px)
  - 2-3 column adjustments
  - Responsive typography scaling
  - Touch-friendly spacing

MOBILE (<768px)
  - 2-column product grids
  - 1-column form stacking
  - Simplified navigation
  - Touch targets 44x44px minimum
```
