## 📞 SUPPORT & TROUBLESHOOTING

### If Migration Fails

```bash
# 1. Check backup exists
ls db_backup_*.sqlite3

# 2. Stop server
# Ctrl+C

# 3. Restore backup
cp db_backup_YYYYMMDD_HHMMSS.sqlite3 db.sqlite3

# 4. Rollback migrations
python manage.py migrate master_products 0001
python manage.py migrate master_brands zero

# 5. Restart and investigate
python manage.py runserver
```

### Common Errors

**Error**: `no such table: master_brands_brandprofile`
- Cause: Migration order issue
- Fix: Check migration dependencies

**Error**: `duplicate key value`
- Cause: nib_or_ktp already exists
- Fix: Check for duplicates in data

**Error**: `foreign key constraint failed`
- Cause: Related data issue
- Fix: Verify orphaned records before migration
